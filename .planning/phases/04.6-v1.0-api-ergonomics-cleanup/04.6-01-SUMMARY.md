## Accomplishments

- Added `pub use scene::{Scene, SceneBuilder}` to `src/lib.rs` — users can now write `use eidos::SceneBuilder` without module qualification
- Added `pub use primitives::{Arrow, Bezier, Circle, Line, Primitive, Rect, Text}` — all seven primitive types accessible at crate root
- Added sub-path re-exports for all four State types: `CircleState`, `RectState`, `LineState`, `TextState` — first-class public API alongside other dataviz types
- Deleted the deprecated `encode_to_mp4` function from `src/svg_gen.rs` (105-line block including doc comment and function body); zero callers confirmed via grep
- Fixed orphaned doc comment on `encode_to_mp4_animated` that referenced the now-deleted `encode_to_mp4`

- Deleted the entire `encode_to_mp4` doc comment block (lines 96-105) as it described Phase 1 static-frame behavior belonging only to the deleted function
- The `encode_to_mp4_animated` doc comment was simplified to be self-contained: "Calls `frame_fn(frame_index)` once per frame to generate unique RGBA bytes for each frame"
